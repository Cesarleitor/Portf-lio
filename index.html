SOBRE MIM

Sou estudante de Análise e Desenvolvimento de Sistemas, apaixonado por tecnologia e desenvolvimento de software. Atualmente, estou focado em expandir meus conhecimentos nas principais linguagens e ferramentas utilizadas no mercado, buscando sempre estar atualizado com as tendências e boas práticas da área. Meu estudo atual engloba diversas tecnologias essenciais para a construção de sistemas robustos e escaláveis. Entre elas, destaco a linguagem Java, que tem sido fundamental para o desenvolvimento back-end, e a Linguagem C, que me ajuda a compreender conceitos fundamentais de programação e lógica computacional. Estou também aprofundando meus conhecimentos em Spring Boot, framework que facilita a criação de aplicações web e APIs, promovendo desenvolvimento rápido e seguro. Além do desenvolvimento, dedico atenção especial ao versionamento de código e ao trabalho colaborativo por meio do Git e GitHub, ferramentas indispensáveis para o controle de versões, organização dos projetos e integração em equipes de desenvolvimento. No campo dos bancos de dados, estudo tanto soluções relacionais quanto não relacionais. Tenho experiência com MySQL e PostgreSQL, bancos relacionais amplamente utilizados em aplicações empresariais, onde aplico conceitos de modelagem de dados, consultas SQL avançadas e otimização de performance. Paralelamente, estou explorando bancos NoSQL, que oferecem flexibilidade e escalabilidade para projetos com grandes volumes de dados ou estruturas não convencionais. Recentemente, também comecei a trabalhar com o Drezzle, uma ferramenta que tem me permitido criar aplicações mais modernas e dinâmicas, integrando diversas tecnologias de forma prática. Este portfólio foi criado como uma vitrine da minha evolução técnica e dos projetos que desenvolvo ao longo dessa jornada de aprendizado. Aqui, compartilho minhas experiências, desafios superados e a aplicação prática dos conhecimentos adquiridos. Meu objetivo é não apenas demonstrar o que já conquistei, mas também mostrar meu compromisso contínuo com o crescimento profissional e minha vontade de contribuir com soluções inovadoras no mercado de tecnologia. Estou aberto a novos desafios, estágios e oportunidades que me permitam aplicar e ampliar minhas habilidades, consolidando minha carreira como desenvolvedor de software.

PROJETO E-COMMERCE

INICIADO

Projeto Ecommerce – Desenvolvimento Iniciado Este projeto de ecommerce está em fase inicial de desenvolvimento e tem como objetivo criar uma plataforma robusta e eficiente para compra e venda online. O foco principal é oferecer uma experiência de usuário fluida, segura e intuitiva, com funcionalidades que atendam tanto os clientes quanto os administradores da loja. As tecnologias que estou utilizando incluem: Backend: Java com Spring Boot para construir uma API RESTful segura e escalável; Banco de Dados: MySQL para armazenar informações estruturadas como produtos, usuários, pedidos e pagamentos; Controle de Versão: Git e GitHub para gerenciar o código e versionar o projeto de forma organizada; Frontend: Em desenvolvimento, com possibilidade de usar React ou outra biblioteca para criar uma interface dinâmica; Funcionalidades previstas: cadastro de usuários, listagem e busca de produtos, carrinho de compras, sistema de pagamento e painel administrativo. Estou usando esse projeto para aplicar meus conhecimentos em programação, banco de dados, arquitetura de software e boas práticas de desenvolvimento. A ideia é evoluir constantemente a aplicação, incluindo recursos como autenticação, integração com APIs de pagamento e otimização da performance. Este ecommerce será um diferencial no meu portfólio, mostrando a capacidade de iniciar, estruturar e desenvolver um sistema completo com foco no mercado real.

Portfólio

O JOGO VIROU

07/08/2025

É com muita alegria e gratidão que anuncio o lançamento oficial do meu novo single, “O Jogo Virou”, disponível a partir do dia 07 de agosto de 2025. Esse projeto é fruto de uma jornada de superação, fé e transformação. Cada faixa carrega uma parte da minha história, das lutas enfrentadas, das vitórias conquistadas e da missão que carrego de usar a música como instrumento de mudança. “O Jogo Virou” é mais do que um título — é uma declaração de vida. Um grito de quem saiu do fundo e hoje caminha com propósito. São letras que falam de dor, mas também de esperança. De queda, mas também de recomeço. De rua, mas também de redenção. O álbum traz batidas marcantes, rimas conscientes e participações que fortalecem ainda mais essa caminhada. Tudo pensado para alcançar corações, inspirar mentes e glorificar Aquele que realmente vira o jogo: Cristo. Disponível em todas as plataformas digitais e nos eventos ao vivo. Ouça, compartilhe e sinta essa verdade: o jogo virou, e a vitória vem do alto.
